#start

#How To start Using Web Apps

This is the documentation for using our web app basic installation using laravel 5.8

Instalation

  1. Download Web Apps Extract Folder Zip in your htdocs folder.
  2. Install Composer.
  3. Open Web Apps Folder on your htdocs in Terminal and run composer update. for update dependecies.
  4. Create a New Database for Web Apps.
  5. Connect to your database by setting the .env file.
  6. Run php artisan migrate:fresh --seed

#Frontend Ajax Get Data

This code show you how to get and display data from API. using ajax jquery.

    $(document).ready(function() {
        generateDataCrud();
    });
    
    function generateDataCrud() {
            let elementDataCrud = new Array();
        
        $.ajax({
            type: "GET",
            url: "/get-crud",
            contentType: "application/json",
            crossDomain: true,
            async: false,
            success: function (data) {
                elementDataCrud = data;
            },
                    dataType: "json"
        });	
    }

    $('#formDataCrud').submit(function (event) {
